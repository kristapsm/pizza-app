Instructions for running valhalla bootstrap

1. Run bootstrap.bat as Administrator (Right click on bootstrap.bat and Run as administrator)
2. Needed files will begin to download from server and extract
3. All files will be saved in C:\valhalla-bootstrap\
4. Follow on-screen instructions when needed
5. Installation wizard for each tool might start with delay, so be patient and wait till it does
6. After bootstrap is complete, JAVA_HOME, GRADLE_HOME and PATH will be set and ready to use (depends whether you chose to set Env.variables or not)

Console output can be found in console.log
